我们知道

gf of k-cores = 

$P(q^k)^k / P(q)$

如果这里k取成分数，则是最近流行的fractional partition function.

如果k是负数呢？很遗憾。在k=-1的例子中，

$gf={1\over {\prod_{n\geq 0 } (1-q^n)(1-q^{-n}) } }$

这样的问题是分母不是有意义的形式幂级数——看常数项吧，趋于无穷，有无数多的q的正负次幂可以抵消。

这样还能定义有意义的(-n)-core吗？
