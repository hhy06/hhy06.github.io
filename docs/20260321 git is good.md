# git is good 

I am increasingly relying on git to transfer stuff between devices. 

For example, i may create an rss-watcher (via opencode) in my phone. I work on one phone via ssh, and transfer it to my other phone to deploy.

I don't even need github to relay. In fact I don't have gh on the phone and I am too lazy to create the repo manually on the webpage. 

What I do is this:

    >git clone ssh://hhy06@192.168.1.27:2022/home/hhy06/rss_monitor/

which is officially the fastest way to transfer files between multiple os/devices.
